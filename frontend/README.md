# Restruct Warp Frontend

This is the frontend for the Restruct Warp Website Generator, built with React and Vite.

## Theme: iOS 26
The design follows a futuristic "iOS 26" aesthetic with:
- **Glassmorphism**: Extensive use of `backdrop-filter: blur()` and semi-transparent backgrounds.
- **Minimalism**: Clean typography, ample whitespace, and focused interactions.
- **Animations**: Smooth fade-ins and scale effects.
- **Dark Mode**: Deep blacks and dark grays inspired by Apple's Pro interfaces.

## Features
- **Authentication**: Seamless Login/Register flow.
- **AI Generation**: Prompt the AI to build websites.
- **Live Preview**: Instantly view the generated HTML code rendered in an iframe.
- **Download**: Export the generated website as an `.html` file.

## Setup
1. Ensure the backend is running at `http://127.0.0.1:8000`.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.
4. Open `http://localhost:5173` in your browser.

## Tech Stack
- React (Vite)
- Lucide React (Icons)
- Vanilla CSS (Custom Design System)
